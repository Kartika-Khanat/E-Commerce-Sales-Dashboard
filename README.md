# E-Commerce Sales Dashboard

## Project Overview
This project is an interactive E-Commerce Sales Dashboard created using Power BI.  
The dashboard helps analyze sales performance, profit, quantity, customer insights, and category-wise performance.

## Tools Used
- Power BI
- Excel / CSV Dataset

## Dashboard Features
- Total Sales Analysis
- Profit Analysis
- Quantity Analysis
- Customer Insights
- Category-wise Performance
- Interactive Filters and Visualizations

## Files Included
- E-Commerce Sales Dashboard.pbix
- Dataset File (.csv / .xlsx)
- Dashboard Screenshot
